### 📝 Notes
- `@as`: Safe widening (small -> large).
- `@intCast`: Risky narrowing (large -> small).
- `@bitCast`: Reinterpret bits (same size, e.g., u8 -> i8).
- **Type Logic**: Use `@bitCast` when you need the same bits to mean something else (e.g., treating a `u8` offset as a signed `i8` for backwards jumps). Use `@as` when you want to change the "container" size while keeping the same value.
- **Root Context**: Always run `zig build run` from the project root. `FileNotFound` occurs if run from `src/` because relative paths (e.g., `roms/`) fail.
- **Stub Logic**: Use `=> {},` instead of `=> return,` for unimplemented opcodes. `return` exits the entire `step()` function, skipping the Program Counter increment at the bottom and causing an infinite loop on the same instruction.
